Attributreihenfolge

HTML-Attribute sollten in der folgenden Ordnung erscheinen, um ihre Lesbarkeit zu erleichtern.

  • id
  • name
  • class
{% highlight html %} Beispiel link {% endhighlight %}
{% highlight haml %} %a#id.class{ name: 'foo', href: '#', data: { content: 'foo' } } Beispiel link %input.form-control{ type: 'text' } %img{ alt: 'foo', src: 'bar.png' } {% endhighlight %}
