Goldene Regel

Halte dich bestmöglichst an die folgenden Regeln. Erstelle für etwaige Änderungen und / oder Ergänzungen ein Issue im Github Repository. Erstellte Issues werden im nächsten Frontend-Chapter besprochen.

Jede Zeile Code sollte so aussehen, als wäre sie von Menschen geschrieben. Gleich, wie viele Personen wirklich daran gearbeitet haben.
