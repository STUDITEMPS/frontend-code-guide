Goldene Regel

Halte dich immer an diese definierten Regeln. Erstelle für etwaige Ergänzungen ein Issue im Github Repository. Erstellte Issues werden im nächsten Fach-Chapter besprochen.

Jede Zeile Code sollte so aussehen, als wenn sie von einer Person geschrieben wurde, egal, wieviele wirklich daran gearbeitet haben.
