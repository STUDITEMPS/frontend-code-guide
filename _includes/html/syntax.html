HMTL Syntax

  • Benutze „Soft-Tabs” mit 2 Leerzeichen um zu garantieren, dass der Code in allen Umgebungen gleich gerendert wird.
  • Verschachtelte Elemente werden einfach (2 Leerzeichen) eingerückt.
  • Benutze keine abschließenden Schrägstriche in selbstschließenden Elementen. Diese sind laut HTML5-Spezifikationen optional.
  • Vergesse nicht die schließenden Tags (z.B. </li> oder </body>).

Hello, world!

{% highlight haml %} !!! %html %head %title Seitentitel %body %img{ alt: 'Company', src: 'images/company-logo.png'} %h1 Hello, world! {% endhighlight %}
